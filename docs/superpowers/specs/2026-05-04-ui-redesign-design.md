# 民宿管家 UI 重设计规范

**日期：** 2026-05-04  
**目标用户：** 四五十岁女性民宿主理人，有审美、追求生活品质  
**设计原则：** 移动端优先，有温度感，不要工具感后台风格

---

## 一、设计风格

**方向：墨绿静谧**

参考感觉：日式杂志排版、小红书克制简洁、Airbnb / 木鸟民宿类 App。  
不要纯蓝/纯红工具色，不要"一眼看上去是商家后台"的感觉。

---

## 七、页面规范

### 7.1 新增预订

- 页面内边距：`16px`
- 每个表单项：标签（9–10px / 600 / `#9a8e85` / letter-spacing）+ 输入框
- 输入框：白色背景，`border-radius: 12px`，`border: 1.5px solid #e2ddd5`
- 聚焦/已填状态：`border-color: #3c6652`
- 字段顺序：选择房源 → 客人姓名 → 入住/退房日期 → 收款金额 → 来源渠道 → 备注
- 提交按钮：全宽，`background: #3c6652`，高度 50px，`border-radius: 14px`，`box-shadow: 0 4px 16px rgba(60,102,82,0.28)`

### 7.2 排期日历（甘特图）

**布局结构：**
- 月份导航栏：左右箭头 + 月份标题 + 预订总数副标题
- 甘特区域：单一滚动容器（`overflow-x: auto; overflow-y: auto`）
  - 表头行：`position: sticky; top: 0`（随容器横向滚动）
  - 日期列：`position: sticky; left: 0`（纵向滚动时固定）
  - 内容区：`min-width: 100%`，超出4间房时向右溢出可横滑

**表头列（每间房）：**
- 房间名（6.5px / 700，单行不换行）
- 房间专属色条（3px 高，全列宽，紧贴文字下方）

**列宽：**
- 日期列：34px（sticky）
- 房间列：当房源数 ≤ 4 时，`(容器宽 - 34px) ÷ 房源数`（均分填满），最小 44px；房源数 > 4 时固定 44px，超出宽度向右横滑
- 行高（`row-h`）：26px

**色块（Booking Block）：**
- 绝对定位，浮在格线网格之上
- 颜色 = 该房源专属色（与预订状态无关）
- 圆角：`5–6px`（整块统一，不分顶中底段）
- `top = row-index × row-h + bk-pad`
- `height = nights × row-h - bk-pad × 2`
- 内显客人姓名（6px / 700），溢出隐藏

**今日指示线：**
- 今日行 `border-bottom: 2.5px solid #ff3b30`
- 日期列显示红色小圆点 + 日期数字（红色加粗）
- **页面打开时自动滚动，使今日行出现在视口垂直居中**

**图例：** 底部一行，各房间色条 + 房间名简称 + 今日红线说明

**预订状态：** 只有「已确认」一种状态，无「待确认」。

### 7.3 预订记录

- 顶部搜索栏：白色圆角输入框，Lucide `Search` 图标，placeholder "搜索客人姓名…"
- 预订卡片（轻盈悬浮型）：
  - 白色背景，`border-radius: 14px`，`box-shadow: 0 2px 16px rgba(42,74,58,0.08)`
  - 左上：客人姓名（14px / 700）+ 房间名·晚数（9px / `#9a8e85`）
  - 右上：金额（15px / 800 / `#3c6652`）+ 日期范围（9px / `#9a8e85`）
  - 底部分割线后：「已确认」绿色胶囊 Tag + 来源渠道（9px / `#9a8e85`）
- 按入住日期倒序排列
- 支持月份筛选

### 7.4 房源管理

- 每间房源卡片：
  - 顶部封面图区（64px 高渐变色块，使用房间专属色渐变）
  - 卡片体：房间名（13px / 700）+ 日租/月租价格（9px / `#9a8e85`）
  - 底部：「上架中」绿色胶囊 + 房型标签
  - 右上角编辑图标（Lucide `Edit2` / `#9a8e85`）
- 底部「添加房源」虚线框按钮

### 7.5 收益明细

**月度汇总卡（深绿底）：**
- 背景：`#2a4a3a`，`border-radius: 16px`
- 月份导航（左右切换）集成在卡内顶部
- 净收益大数字：28px / 800 / `#b8ddc8`（浅绿）
- 总收入 / 总成本：两格并排，`background: rgba(255,255,255,0.1)`
- 成本构成细项：分割线后细列（10px）：房租、物业水电暖气、维修采购、保洁

**本月入住率（每间房一张卡）：**
- 房间色圆点（7px）+ 房间名（11px / 700）
- 右侧：涨跌幅（↑/↓，绿色或暖棕，不用红色）+ 本月入住率（14px / 800）
- 本月进度条：7px 高，房间专属色
- 已预订 X 天 / 共 X 天（8px / `#9a8e85`）
- 上月对比进度条：4px 高，`#d5cfc8` 灰色 + 上月百分比标注

**各房间明细（可展开卡）：**
- 卡片顶部 3px 房间专属色条
- 三格数据：本月收入（绿色）/ 本月成本（暖棕）/ 净收益
- 「展开支出明细」按钮：`border: 1px solid #e2ddd5`，轻描边风格
- 展开后显示：房租 + 各支出条目（日期、类型 Tag、说明、金额）

**导出按钮：**
- 全宽，`background: #3c6652`，带下载图标（Lucide `Download`）
- 文字：「导出本月（YYYY年M月）」

## 八、加载与错误状态

**加载中：**
- 全屏居中，米色背景
- 小图标 + "民宿管家" + "正在连接数据库…"
- 字体风格与顶栏 Logo 一致

**连接失败：**
- 警告图标 + 错误文字（`#2d2926`，非红色）
- 「重试」按钮：`background: #3c6652`

---

## 九、技术约定

- 图标库：**Lucide React**（替换当前 Ant Design Icons 和 emoji）
- UI 组件库：**Ant Design**（保留，但覆盖为新色系）
- 全局 CSS 变量：统一在 `src/index.css` 的 `:root` 中定义
- 最大宽度：`540px`（当前已有，保持不变）
- 安全区：TabBar 使用 `env(safe-area-inset-bottom)`
- 滚动条：全局隐藏（`scrollbar-width: none`）
